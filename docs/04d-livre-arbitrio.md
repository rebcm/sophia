# 04d · Livre Arbítrio — Escolhas, Vozes e Consequências

> *"O Demiurgo desenhou a jaula. Sophia, em sua queda, plantou a
> chave. A chave é a tua liberdade de escolher. E ela funciona
> em ambos os sentidos: pode te abrir, pode te trancar de novo.
> Mas funciona — e isso é o que ele não esperava."*

Este documento detalha como o **livre arbítrio do jogador** se
manifesta no jogo: através de **escolhas em diálogo**, **decisões
de uso de poder**, **interações com anjos, demônios e jinns**, e
**caminhos divergentes** que cada uma destas decisões abre.

A regra ética central: **o objetivo último — a iluminação — é
inevitável**. Mas o **caminho até ela** muda profundamente segundo
as escolhas do jogador. Caminhos de luz são mais curtos; caminhos
de sombra são mais longos e abrem **conteúdo único**; caminhos
equilibrados são os mais profundos.

---

## 1. A tese filosófica

O jogo afirma uma tese que pode soar paradoxal:

> **A liberdade é real. E o destino também é real. Os dois
> coexistem porque ambos vivem em camadas diferentes do mesmo
> ser.**

Em outras palavras: **a iluminação é inevitável** porque é o
**estado natural** da centelha humana. Mas **quando** e **como**
chegar lá — quantas vidas, quantas dores, quantos atalhos —
depende **inteiramente** de cada escolha.

**Ninguém deixa de chegar. Todos chegam. Mas em ritmos
radicalmente diferentes.**

---

## 2. Os três alinhamentos

Cada escolha do jogador acumula em **um** dos três medidores:

```
              ┌──────────────┐
              │     LUZ      │   ← compaixão, escuta, doação
              └──────┬───────┘
                     │
            ╔════════════════╗
            ║   EQUILÍBRIO   ║   ← discernimento, presença
            ╚════════════════╝
                     │
              ┌──────┴───────┐
              │   SOMBRA     │   ← controle, medo, manipulação
              └──────────────┘
```

### **Luz**

Escolhas que servem **outros sem expectativa de retorno**, que
**escutam antes de falar**, que **dispensam o próprio brilho**.

### **Sombra**

Escolhas que **manipulam**, que **escondem**, que **usam o poder
para conquistar mais poder**, que **cobram favor por bondade**.

> **Importante:** "Sombra" não é "mal". É **força viva mal
> direcionada**. Não há "vilão" como caminho. A Sombra do jogador
> ainda termina iluminada — só com mais voltas.

### **Equilíbrio**

Escolhas conscientes de **discernimento** — nem doação cega, nem
controle. Reconhecer **quando** dar e **quando** segurar. É o
caminho **mais difícil** porque exige **maturidade espiritual**
e **timing**.

### Visualização

No HUD, a barra de Luz Interior tem **três cores estratificadas**:

- Centro dourado = núcleo de Luz (fixo)
- Borda externa **muda de cor** conforme alinhamento dominante:
  - **Branco-âmbar** = Luz dominante
  - **Violeta-suave** = Equilíbrio dominante
  - **Vermelho-fumaça** = Sombra dominante

## 3. As Quatro Vozes

Em cada escolha-chave, o jogador **escuta vozes** — vozes
internas que sugerem caminhos. Há **quatro vozes** distintas, e
cada uma fala em um momento próprio.

### Voz 1 · O Anjo

**Sussurra a verdade simples e desconfortável.**

- Costuma sugerir o caminho **mais difícil emocionalmente** mas
  **mais leve espiritualmente**.
- *"Cala-te. Apenas escuta."*
- *"Não precisa fazer nada. Apenas senta com ela."*
- *"Solta isso. Tu não precisas."*

Visualmente: **luz dourada quente** que pulsa no canto direito da
tela.

### Voz 2 · O Demônio

**Sussurra atalhos, justificativas, vinganças.**

- Costuma sugerir o caminho **mais cômodo agora** mas com
  **dívida adiante**.
- *"Tu mereces."*
- *"Eles te devem."*
- *"Pegue agora. Decide depois."*

**Crucial:** o demônio **não mente**. Ele **diz a verdade
parcial**. Suas sugestões funcionam — mas em **prazo curto**.
**No longo prazo**, custam.

Visualmente: **fumaça escura** que se enrola no canto esquerdo
da tela. Quando o jogador escolhe, a fumaça **se acomoda no
peito** — aparecerá na próxima cutscene de espelho.

### Voz 3 · O Jinn

**Pergunta, propõe, negocia, ri.**

- Costuma sugerir o caminho **lateral** — fora do binário
  luz/sombra.
- *"Por que escolher entre os dois? Faça uma terceira coisa."*
- *"E se tu desses isso para mim em troca daquilo?"*
- *"O preço é alto. Mas há um preço, sim."*

Visualmente: **chama dourada com bordas azuis** que paira no alto
da tela. Tom: jocoso, esperto, neutro.

### Voz 4 · Sophia

**Não fala diretamente. Sopra silêncio.**

A Sussurrante de Sophia **não toma partido nas escolhas**. Mas
quando o jogador hesita por mais de 30 segundos, ela
**sussurra**:

- *"Tu já sabes. Apenas esquece quem te disse."*
- *"Nenhuma escolha te perde. Algumas te alongam."*
- *"O que sentes agora — escuta isso."*

Sophia **nunca diz a resposta**. Ela apenas **devolve o jogador
a si mesmo**.

## 4. Anatomia de uma escolha

Cada escolha-chave (e há **dezenas** ao longo do jogo) tem a
mesma estrutura:

```
┌─────────────────────────────────────────────────────┐
│ Situação dramática (cena, NPC, contexto)            │
├─────────────────────────────────────────────────────┤
│ As Vozes começam a sussurrar (3-4)                  │
│   Anjo: ...                                          │
│   Demônio: ...                                       │
│   Jinn: ... (às vezes)                               │
│   Sophia: (silêncio com pequena dica) (após 30s)    │
├─────────────────────────────────────────────────────┤
│ Opções de escolha (3 ou 4):                         │
│   1. [Luz]                                           │
│   2. [Sombra]                                        │
│   3. [Equilíbrio]                                    │
│   4. [Pacto-Jinn] (às vezes)                        │
├─────────────────────────────────────────────────────┤
│ Consequência imediata                                │
│ + Eco para o futuro                                  │
└─────────────────────────────────────────────────────┘
```

### Exemplo concreto · Atlântida, capítulo 4 (Centelha da Palavra)

Você está diante do **Patriarca-Cristal**. Você acabou de tocar
três cristais que mostram leis racha-das. Agora ele te oferece o
martelo dourado para **destruí-las todas** e libertar o templo de
uma vez.

**As vozes:**
- **Anjo:** *"Não destruas. Nomeia. As leis quebradas são pessoas
  que te seguem — quebrá-las é quebrá-las."*
- **Demônio:** *"Pegue o martelo. Acabe com isso. Eles não merecem
  pensar tanto. Faça pelo bem deles."*
- **Jinn:** *"Eu te ofereço uma quarta opção: dá-me o martelo, e
  eu nomeio cada lei pelo seu nome verdadeiro. Em troca, tu
  carregarás minha chama por uma esfera."*

**Opções:**
1. **[Luz]** Aceitar o martelo, mas usá-lo apenas para nomear,
   não destruir. *(+1 Luz; o templo se cura lentamente)*
2. **[Sombra]** Pegar o martelo e destruir tudo de uma vez.
   *(+2 Sombra; o templo desaba imediatamente, Atlântida afunda
   um pouco mais rápido)*
3. **[Equilíbrio]** Recusar o martelo. Sentar-se diante das
   leis e nomeá-las **com a voz**. *(+1 Equilíbrio; demora 5
   minutos de gameplay extra mas desbloqueia uma cutscene rara
   de Sophia)*
4. **[Pacto]** Aceitar a oferta do Jinn. *(Carrega chama do
   Jinn por uma esfera; ganha vantagem temporária mas tem
   compromisso pendente)*

**Eco:**
- Escolha 1 (Luz): Patriarca-Cristal aparecerá em Lemúria como
  Aliado Acordado.
- Escolha 2 (Sombra): a Criança-Refugiada parte mais cedo no
  navio (faltam Sleepers desconhecidos para acordar em Egito
  posteriormente).
- Escolha 3 (Equilíbrio): o jogador desbloqueia o **modo de
  cantar leis** que pode ser usado em outras dimensões.
- Escolha 4 (Pacto): o jogador deve cumprir promessa ao Jinn em
  até 3 horas in-game.

## 5. Os efeitos do alinhamento

### Quando Luz domina (Luz > 70% do total acumulado)

**Vantagens:**
- Anjos aparecem **com mais frequência**
- Sleepers são **mais fáceis** de despertar
- Aliados Acordados se reúnem **mais rápido**
- Aeons-Mestres aparecem **espontaneamente** em algumas cenas
- Animais selvagens **seguem** o jogador em paz
- Algumas portas se abrem **só pela presença**

**Desvantagens:**
- Demônios **conspiram com mais agressividade** (pelo contraste)
- Algumas dimensões "duras" (ex: Catedral-Trono interna) **resistem**
- Conteúdo de **Sombra** fica **bloqueado**

### Quando Sombra domina (Sombra > 70%)

**Vantagens:**
- Acesso a **dimensões sombrias secretas** (ex: o **Reino Invertido**,
  uma versão espelhada de Atlântida onde os Arcontes estão soltos)
- Acesso a **Aliados sombrios** — figuras complexas como **a
  Bruxa-de-Ratanabá**, **o Pirata-Mensageiro** (de El Dorado) que
  só ajudam quem fez escolhas duras
- **Habilidades-de-Sombra** únicas: **Olhar-Que-Conhece-Segredos**
  (lê pensamentos hostis), **Voz-Que-Manda** (Sleepers obedecem
  brevemente)
- **Conteúdo narrativo único** — diálogos especiais com
  Yaldabaoth no clímax

**Desvantagens:**
- Anjos **não aparecem mais** (até o jogador retornar ao
  Equilíbrio)
- Aliados Acordados **se afastam** — alguns podem deixar o
  Conselho
- Sleepers ficam **mais difíceis** de despertar — exigem mais
  atenção
- O ending **canônico (B)** fica **mais distante** — tem que
  passar por uma fase **extra** de "expiação" (não é punição:
  é **trabalho de reabsorver as sombras** acumuladas)
- **Não pode entrar** no Coração Quieto (XVI) sem balancear

### Quando Equilíbrio domina (Equilíbrio > 50%, ambos os outros
< 30%)

**Vantagens:**
- **Acesso completo** a todo o conteúdo do jogo — dimensões,
  Aliados, vozes
- **Discernimento** automático em escolhas posteriores (a tela
  mostra **prévia das consequências** antes de decidir)
- **Habilidades-de-Equilíbrio** únicas: **Ver-Antes-de-Agir**
  (visualizar consequência de uma ação antes de executá-la);
  **Espelho-Honesto** (mostrar a alguém sua própria sombra sem
  julgar)
- **Caminho mais profundo** ao Pleroma — Sophia fica **mais
  visível** desde cedo

**Desvantagens:**
- **Mais lento.** Equilíbrio exige **escutar mais antes de
  decidir**. O jogador "perde" tempo de gameplay.
- **Conteúdo exclusivo de Luz** ou **de Sombra** fica
  **fragmentário** — só vê pedaços.

### Endings disponíveis por alinhamento

| Alinhamento dominante | Endings naturais |
|------------------------|------------------|
| **Luz pura** | A (Ascensão), B (Bodhisattva canônico), C (Apocatástase, se completou tudo) |
| **Sombra dominante** | E (Bardo Direto), F (Esquecimento), B com fase extra de expiação |
| **Equilíbrio dominante** | C (Apocatástase) e D (Aeon-Mestre) preferenciais |

**Mas:** *qualquer alinhamento* pode chegar a *qualquer ending*
**desde que o jogador queira**. Sombra dominante pode chegar a
ending C, mas exige **mais horas** e **fase extra**. Equilíbrio
pode chegar a ending A, mas exige **renunciar gradualmente** ao
caminho discernido. **Liberdade absoluta. Apenas custos
diferentes.**

## 6. Os ecos cumulativos

Cada escolha **ecoa** no resto do jogo. Há **três tipos de eco**:

### Eco Imediato

Acontece **na próxima cena** após a escolha.

- Acolher o demônio interno → Sleeper próximo desperta sozinho
- Manipular um Sleeper → ele acorda mas **não confia** no jogador
- Aceitar pacto com Jinn → chama dele orbita ao redor

### Eco Médio

Acontece **dentro da mesma dimensão / civilização**.

- Recusar martelo do Patriarca-Cristal → 5 minutos depois, uma
  cabana se abre revelando **o Sacerdote-Atlante** (Sleeper
  raro)
- Destruir leis com martelo → Atlântida afunda **mais
  rapidamente**

### Eco Longo

Acontece **muito depois**, em outra dimensão ou no Pleroma.

- A Criança-Refugiada que partiu cedo → não está disponível em
  Egito como Sleeper (perda permanente)
- Acolher 7 demônios → no Pleroma, um **Coro extra de almas**
  aparece, agradecendo
- Pactuar com 7 Jinns → no ending D, o jogador é convidado para
  um conselho intersticial dos Jinns como Aeon-Mestre

## 7. Anjos e Demônios em ação

Na prática de jogo, **anjos e demônios não são apenas conselheiros**
— eles **influenciam o gameplay**.

### Quando um anjo aparece (sem ser chamado)

- A tela ganha **luz suave**.
- Sons hostis **abafam-se**.
- Sleepers próximos **acalmam-se**.
- O jogador recebe **+0.3 Luz Interior** apenas pela presença.

### Quando um demônio aparece (geralmente após escolha de Sombra)

- A tela ganha **vinheta escurecida**.
- O som **fica mais agudo**.
- Sleepers próximos **desconfiam** (não fogem, mas hesitam).
- O jogador **não perde Luz** automaticamente, mas **a próxima
  escolha** vem com **uma quarta voz fantasma** sussurrando
  sugestões manipuladoras.

### Como reverter Sombra acumulada

A qualquer momento, o jogador pode **encontrar um anjo
voluntariamente** indo a um santuário e **rezando** (animação de
ajoelhar + segurar tecla por 60s). O anjo aparece, escuta
silenciosamente, e **dá ao jogador a oportunidade** de **acolher
um demônio interno** (ver
[03c-anjos-demonios-jinns.md](03c-anjos-demonios-jinns.md), seção
3).

**Cada demônio acolhido = -1 ponto de Sombra + +0.5 Luz Interior.**

> Esta é a **redenção mecanizada** — não como punição, mas como
> **caminho sempre disponível**. **Ninguém é deixado para trás.**

## 9. Sistema de Rastreamento

### No Codex (UI)

Aba **"Caminhos"** mostra:

- Barra dos três alinhamentos (Luz / Equilíbrio / Sombra)
- Lista de **escolhas-chave já feitas** (com resumo de uma linha
  cada)
- **Ecos pendentes** (mudanças que vão acontecer "em breve")
- **Pactos pendentes** com Jinns

### Aba "Espelho"

Visualização **artística** do jogador atual:

- **Avatar** com aspectos visuais que refletem alinhamento
  (luz dourada vs sombra fumacenta vs violeta-equilíbrio)
- **Coroa** se Aeon Humano (após Demiurgo)
- **Cicatrizes** se aceitou pactos especiais com Jinns
- **Asas** se aliado dos anjos

### Aba "Sombras"

Lista dos demônios acolhidos, com a **frase** que cada um disse
ao se dissolver. Funciona como **diário psicológico do jogador**.

## 10. A regra inviolável da redenção

Importa repetir: **nenhum jogador é punido por escolhas de
sombra**. O jogo **não tem game-over por imoralidade**. **Não
existe "alinhamento ruim irrecuperável".** A qualquer momento,
**qualquer caminho** está disponível.

> *"Se ele escolhe Sombra durante o jogo todo, ainda vai chegar
> à Mônada. Talvez precise jogar 50% mais. Talvez tenha que
> acolher 30 sombras em vez de 7. Talvez veja Sophia chorar
> mais vezes. Mas vai chegar. Esta é a regra do livre arbítrio:
> a porta nunca se tranca para sempre."*

## 11. Comparação com outros jogos

Para alinhamento de expectativa de mercado, *Sophia* segue uma
linha diferente de:

- **Mass Effect / Fable / KotOR** (paragon vs renegade) — o jogo
  trata escolhas como **moralidade binária**. *Sophia* usa
  **três eixos**, e Sombra **não é mau** — é energia mal
  direcionada.
- **Disco Elysium** — escolhas mais filosóficas e abertas.
  *Sophia* segue parcialmente esta tradição, com mais
  consequências espirituais explícitas.
- **Detroit: Become Human** — branching cinemático denso.
  *Sophia* compartilha esta densidade mas com escala maior
  (dimensões inteiras se abrem/fecham, não só cenas).
- **Undertale** — pacifismo total como caminho viável. *Sophia*
  herda diretamente este princípio: **toda escolha pode ser
  pacífica** e **acolhedora**.

## 12. Implementação técnica (resumo)

### State

```typescript
// src/state/gameStore.ts (a expandir)
interface AlignmentState {
  light: number;        // 0-100
  shadow: number;       // 0-100
  balance: number;      // 0-100
  totalChoicesMade: number;
  choices: ChoiceMade[];
  jinnPacts: JinnPact[];
  shadowsAccepted: ShadowAccepted[];
}

interface ChoiceMade {
  id: string;
  question: string;
  optionChosen: 'light' | 'shadow' | 'balance' | 'pact';
  consequence: string;  // resumo curto
  timestamp: number;
}

interface JinnPact {
  jinnId: string;
  bargain: string;
  benefit: string;
  obligation: string;
  fulfilled: boolean;
}

interface ShadowAccepted {
  shadowId: string;
  recognizedAs: string;
  insightGained: string;
}
```

### Voice system

Cada escolha-chave tem **dados estruturados**:

```typescript
interface KeyChoice {
  id: string;
  scene: string;                      // localização in-game
  context: string;                    // para o leitor de tela
  voices: {
    angel?: string;
    demon?: string;
    jinn?: string;
    sophia?: string;  // após hesitação
  };
  options: ChoiceOption[];
  echoes: Echo[];
}

interface ChoiceOption {
  label: string;
  alignment: 'light' | 'shadow' | 'balance' | 'pact';
  pactData?: JinnPactData;
  immediateEffect: string;
}

interface Echo {
  trigger: 'immediate' | 'medium' | 'long';
  duration?: number;  // em horas in-game
  effect: string;
}
```

### Persistência

Alinhamento e escolhas **sempre persistem**, mesmo entre
reencarnações. **A única coisa que zera** é se o jogador
**explicitamente** escolhe **resetar a vida** no Codex (ação
opcional, não recomendada — perde Memórias Anteriores).

## 13. Resumo filosófico

> *"Tu és livre. Eis a frase mais perigosa do cosmos. Porque
> liberdade é peso. Quem és é o que escolhes — não uma vez,
> mas em cada gesto, em cada palavra, em cada silêncio. Sophia
> te dá a chave. Não te dá o caminho. O caminho é teu."*

O sistema de livre arbítrio em *Sophia* é a **mais explicitamente
gnóstica** de todas as mecânicas: porque para os gnósticos, **a
salvação não é nem por crença nem por obra — é por escolha
consciente**. *Gnosis* é **escolha de olhar**. O jogo encarna
isso em código.
